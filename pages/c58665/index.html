        markdown 字母、数字间自动插入空格（textinit工具）

        # markdown 字母、数字间自动插入空格（textinit 工具）

        # 介绍

        Textlint (opens new window)是一个用 JavaScript 编写的开源文本列表实用程序。The pluggable linting tool for text and markdown.

        # 使用

        1. 创建项目

          npm init --yes
          1

        2. 安装 textlint

          npm install --save-dev textlint
          1

        3. 安装 textlint 规则

          npm install --save-dev textlint-rule-no-todo
          1

        4. 创建.textlintrc 文件

          npx textlint --init
          1

        5. 运行 textlint

          npx textlint README.md
          1

        6. 自动修复问题

          npx textlint --fix README.md
          1

        # 常用规则

        Collection-of-textlint-rule (opens new window)

        # 推荐规则

        # textlint-rule-alive-link

        检查所有链接是否可用。

        # textlint-rule-ja-space-between-half-and-full-width (opens new window)

        检测半角字符和全角字符之间是否有空格

        1. 安装

          npm install --save-dev textlint-rule-ja-space-between-half-and-full-width
          1

        2. 使用

          在.textlintrc 中添加（推荐）

          {
    "rules": {
        "ja-space-between-half-and-full-width": {
            "space": "never"
        }
    }
}
          1

        3. 选项

          • space: "always" || "never" || string[]

            • 默认值: never

            • 总是空格（always）或不空格（never）

            • 也可以使用 Array 格式: [alphabets,numbers，punctuation]

              • 只指定想要的对象
              • 例如，如果希望将数字和标点符号(、。)视为例外:["alphabets"]

          • ...

          {
    "rules": {
        "ja-space-between-half-and-full-width": {
            "space": "always"
        }
    }
}

        # textlint-rule-zh-half-and-full-width-bracket (opens new window)

        检测是否正确地使用了全角或半角括号。默认情况下，半角和全角之间没有空格。("never")

        1. 安装

          npm install --save-dev textlint-rule-zh-half-and-full-width-bracket
          1

        2. 使用

          在.textlintrc 中添加（推荐）

          {
  "rules": {
    "zh-half-and-full-width-bracket": true
  }
}
          1
          2
          3
          4
          5

        3. 选项

          • "halfWidth"：一律使用半角括号
          • "fullWidth"：一律使用全角括号
          • "mixed"：括号内有中文的情况下，使用中文全角括号；括号内是全英文、数字的情况下，使用英文半角括号
          {
  "rules": {
    "zh-half-and-full-width-bracket": {
    // type of brackets you want
    // 选择你需要的括号类型
      "bracket": "halfWidth"
    }
  }
}
