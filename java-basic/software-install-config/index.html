# 配置环境变量

变量名：JAVA_HOME
变量值：JDK的安装路径
1
2

  1. 在系统变量 Path 环境变量中添加：%JAVA_HOME%\bin
  2. java -version

# maven

# 配置环境变量

  1. 添加系统变量 MAVEN_HOME
变量名：MAVEN_HOME
变量值：maven的安装路径或IDEA中的maven路径
1
2
  1. 在系统变量 Path 环境变量中添加：%MAVEN_HOME%\bin
  2. mvn -v

# 中文编码问题

找到 conf 目录下的 logging.properties 文件，将原文件中的 UTF-8 全部替换为 GBK 然后保存即可
